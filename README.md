# 🧘 GMKV Trust Website

A modern full-stack web application for **Gudiyatham Manavalakalai Mandram Trust** to manage members, branches, and activities.

---

## 🌐 Live Features

- 🌍 Multi-language support (English / Tamil)
- 🧘 Membership Registration Form
- 🔐 Admin Login (Firebase Authentication)
- 👤 Member Profile Page
- 📊 Admin Panel (View all members)
- 📩 Email Notifications (EmailJS)
- ☁️ Firebase Firestore Database
- 📱 Fully Responsive Design (Mobile Friendly)

---

## 🚀 Tech Stack

- ⚛️ React + TypeScript
- 🎨 Tailwind CSS
- 🔥 Firebase (Auth + Firestore)
- 📧 EmailJS
- 🌐 React Router
